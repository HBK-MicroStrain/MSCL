# MSCL for Python

MSCL is developed by [MicroStrain](https://www.hbkworld.com/en/Campaign/microstrain-by-hbk)
by HBK. It provides a simple, user-friendly API to interact with MicroStrain
Wireless and Inertial sensors.

## Installation

```shell
pip install pymscl
```

```python
import mscl
```
